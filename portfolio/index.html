Graph Search 2012

User interface prototyping for Graph Search mobile. Graph Search is a natural language search engine that allows people to construct queries like "My friends who live in San Francisco." I looked at many ways to construct queries like this given the constraints of a mobile device.

One approach was to let the user focus on query "nouns" such as people, result type, and places and let the system suggest language that connect concepts together. While working on search I also worked with the voice team to develop integrations with other products, like Messenger. Reimagining Graph Search for iPad.
