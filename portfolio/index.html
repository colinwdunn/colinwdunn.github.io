Graph Search 2012

User interface prototyping for Graph Search mobile. Graph Search is a natural language search engine that lets you search for things like "My friends who live in San Francisco." I did a wide exploration of different ways to build queries given the constraints of a mobile device.

One approach was to let you focus on query "nouns" such as people, result type, and places and let the system suggest connective language. While working on search I also worked with the voice team to develop integrations with other products, like Messenger. Reimagining Graph Search for iPad.
